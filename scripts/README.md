# fbox — 给 AI 看的文件收集箱

## 这是什么？

一个 Windows 桌面小工具，解决一个很具体的痛点：

**你跟 AI 说"帮我看一下这个文件"时，AI 到底该看哪个文件？**

以前你得复制路径、拖进对话、或者手打文件名。
现在你只需要把文件拖进 fbox 的小窗口，AI 自己就能查到。

---

## 工作流

```
你                                AI
  │                                │
  ├─ 把文件拖进 fbox 小窗口 ──────→│
  │                                ├─ fbox list --new
  │                                │  → 拿到文件路径
  │                                │  → 开始处理
  │                                │
  ├─ 又拖了几个文件 ──────────────→│
  │                                ├─ fbox list --new
  │                                │  → 只拿到新增的
  │                                │
  │      （每次 list 自动标记已读，                  │
  │        保证增量永远是新内容）                     │
```

## 核心特性

- **常驻小窗口** — 始终置顶，随时拖入，不打断工作流
- **增量队列** — `fbox list --new` 只看新文件，看过的不重复
- **GUI + CLI 双模式** — 你拖文件用 GUI，AI 查文件用 CLI
- **零配置** — 装完就用，数据存 `%APPDATA%`
- **配套 AI 技能** — 配合 opencode 自动识别，无需手动告知

## 一句话

> 你只管拖文件，AI 自己知道该看什么。

---

## 快速安装

```bash
# 解压 zip → 右键 setup.ps1 → "使用 PowerShell 运行"
# 之后:
fbox list --new    # AI 查新文件
fbox-gui           # 你启动拖拽窗口
```

需要 .NET 9 运行时，或下载自包含版本（免依赖）。
